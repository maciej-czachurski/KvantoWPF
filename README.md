# KvantoWPF

Kvanto is a Windows 11 focused daily task tracker built with WPF.

## Included functionality

- task management with active and archived lists
- per-task stats for tracked time, active days, and Pomodoro progress
- configurable Pomodoro timer with short and long break automation
- monthly calendar view with per-day task blocks and tracked minutes
- date-range reporting with summary cards and lightweight bar charts
- compact always-on-top overlay for the active timer
- system tray actions for opening the app, controlling Pomodoro sessions, and exiting
- global hotkeys for start/pause, skip, and restoring the main window
- session-complete notifications that prompt for the next action

## Build

```bash
dotnet build src/KvantoWPF/KvantoWPF.csproj
```
